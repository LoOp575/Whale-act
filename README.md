# WhaleCopy AI

AI-powered whale wallet tracking and copy trading dashboard (frontend only, dummy data).

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- No backend, no real API, no live trading

## Deploy to Vercel

1. Go to [vercel.com](https://vercel.com) → Import your GitHub repo
2. Framework Preset: Next.js (auto-detected)
3. Root Directory: `.` (default, no change needed)
4. Click Deploy

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
